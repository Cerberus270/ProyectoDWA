World Air Quality Index REST API
API para medir la calidad del aire, en distintas ciudades

El proyecto del Índice Mundial de la Calidad del Aire es un proyecto sin ánimo de lucro iniciado en 2007. Su misión es promover la concienciación de los ciudadanos sobre la contaminación atmosférica y proporcionar una información unificada y mundial sobre la calidad del aire. El proyecto proporciona información transparente sobre la calidad del aire en más de 130 países, cubriendo más de 30.000 estaciones en 2.000 ciudades importantes.

Más información través de estos dos sitios web: aqicn.org y waqi.info.

El índice de calidad del aire (ICA) es una cifra que proporcionan las autoridades de una zona (normalmente urbana) y que refleja las cantidades de contaminantes presentes en el aire.​ Cuando la calidad del aire es buena (tiene pocos contaminantes), las personas pueden respirarlo indefinidamente sin que su salud se vea afectada. Cuando es mala pueden darse irritaciones oculares, todo tipo de problemas respiratorios e incluso cardíacos.

Carbon Interface
API para medir nuestras emisiones de carbono

Usted está aquí porque quiere hacer que su aplicación sea más consciente del clima. Nuestro objetivo en Carbon Interface es facilitar a los desarrolladores la integración de estimaciones de emisiones de carbono con respaldo científico en sus aplicaciones. De este modo, podemos crear un software que reduzca las emisiones de carbono, frenando el cambio climático.   Carbon Interface ofrece dos APIs:

API de estimaciones: La API de estimaciones facilita la generación de estimaciones precisas de las emisiones de los vuelos, los vehículos, el transporte marítimo, el consumo de electricidad y la combustión de combustibles.

API de Carbon Ledger: La API Carbon Ledger facilita la generación de estimaciones precisas de emisiones a partir de transacciones con tarjetas de crédito y débito.

Más información través de este sitio web: aqicn.org

OpenWeatherMap
API que nos muestra las condiciones climaticas, dadas las coordenadas.

OpenWeatherMap es un servicio en línea, propiedad de OpenWeather Ltd, que proporciona datos meteorológicos globales a través de la API, incluyendo datos meteorológicos actuales, previsiones, nowcasts y datos meteorológicos históricos para cualquier ubicación geográfica. La empresa ofrece una previsión de precipitaciones hiperlocal minuto a minuto para cualquier lugar. El modelo de aprendizaje automático convolucional se utiliza para utilizar los servicios de difusión meteorológica y los datos de las estaciones meteorológicas de los aeropuertos, las estaciones de radar en tierra, los satélites meteorológicos, los satélites de teledetección, los METAR y las estaciones meteorológicas automatizadas.

Más información través de este web: openweathermap.org
